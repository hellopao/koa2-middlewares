# koa2-middlewares

a cmd tool to search koa2 middlewares 

## install

```bash
npm install koa2-middlewares
```

## usage

### list

list koa2 middlewares

```bash
koams list 
```

#### options ignore

filter definited koa middlewares

```bash
koams list -i
```

### open

open koa2 middleware's github

```bash
koams open koa-csrf
```
